To create a microservice application for user login using Python, FastAPI, and Redis, here's a brief guide:

Step 1: Set Up the Environment

  1. Install Required Packages:

    • Install FastAPI, Uvicorn (for development server), Redis, and other necessary libraries.
    bash
    pip install fastapi uvicorn redis pydantic

  2. Set Up Redis:

    • Make sure you have Redis installed locally or use a cloud-based Redis service.
    • Redis will be used to manage user sessions (e.g., storing authentication tokens).

Step 2: Define the FastAPI Application

Create a main.py file where you'll define your FastAPI routes and interactions with Redis.

python
from fastapi import FastAPI, HTTPException, Depends
from pydantic import BaseModel
import redis
import hashlib
import os

# Initialize FastAPI and Redis client
app = FastAPI()
redis_client = redis.StrictRedis(host="localhost", port=6379, db=0, decode_responses=True)

# Define a Pydantic model for user login data
class UserLogin(BaseModel):
    username: str
    password: str

# Helper function to hash passwords securely (use bcrypt in production)
def hash_password(password: str) -> str:
    return hashlib.sha256(password.encode('utf-8')).hexdigest()

# Route to handle user login
@app.post("/login")
async def login(user: UserLogin):
    # Check if username exists in the Redis database
    stored_password_hash = redis_client.get(user.username)
    
    if not stored_password_hash:
        raise HTTPException(status_code=400, detail="Invalid username or password")

    # Compare hashed password with stored password
    if stored_password_hash != hash_password(user.password):
        raise HTTPException(status_code=400, detail="Invalid username or password")
    
    # Generate session token (a simple approach using a random value or JWT)
    session_token = os.urandom(16).hex()
    
    # Store session token in Redis for session management
    redis_client.set(session_token, user.username, ex=3600)  # Expiry set to 1 hour

    return {"message": "Login successful", "session_token": session_token}

# Route to check session validity (to be used in other endpoints)
@app.get("/check_session")
async def check_session(token: str):
    username = redis_client.get(token)
    if not username:
        raise HTTPException(status_code=400, detail="Invalid or expired session token")
    return {"message": "Session is valid", "username": username}

Step 3: Running the Application

  1. Run FastAPI:

    • Run the FastAPI application with Uvicorn:
    bash
    uvicorn main:app --reload

    This will start the FastAPI application on http://127.0.0.1:8000.

  • JWT Tokens: For production, consider using JWT for more secure and scalable session management.
  • Password Hashing: Use bcrypt or another hashing library instead of sha256 for better security.

Key Points

6. User Functions (user_api_function.py)

This is the module where user-related logic such as login, registration, and CRUD operations will reside. Redis will be used for session management, while the database handles user data.

python
# user_api_function.py
from sqlalchemy.orm import Session
from .models import User
from .database import get_db
import redis
import hashlib
from .exceptions import UserAlreadyExistsError, InvalidCredentialsError
from pydantic import BaseModel

# Initialize Redis client
redis_client = redis.StrictRedis(host='localhost', port=6379, db=0, decode_responses=True)

# Helper function for password hashing
def hash_password(password: str) -> str:
    return hashlib.sha256(password.encode('utf-8')).hexdigest()

# Register a user
def register_user(db: Session, username: str, password: str):
    # Check if the user already exists
    existing_user = db.query(User).filter(User.username == username).first()
    if existing_user:
        raise UserAlreadyExistsError()

    # Hash the password
    hashed_password = hash_password(password)

    # Create a new user
    new_user = User(username=username, password_hash=hashed_password)
    db.add(new_user)
    db.commit()
    db.refresh(new_user)
    
    return {"message": "User registered successfully"}

# Login a user
def login_user(db: Session, username: str, password: str):
    user = db.query(User).filter(User.username == username).first()
    if not user or user.password_hash != hash_password(password):
        raise InvalidCredentialsError()
    
    # Generate session token and store in Redis
    session_token = hashlib.sha256(username.encode('utf-8')).hexdigest()
    redis_client.set(session_token, username, ex=3600)  # Expiry: 1 hour
    
    return {"message": "Login successful", "session_token": session_token}

# CRUD operations (Example)
def get_user(db: Session, user_id: int):
    return db.query(User).filter(User.id == user_id).first()

def update_user(db: Session, user_id: int, new_username: str, new_password: str):
    user = db.query(User).filter(User.id == user_id).first()
    if user:
        user.username = new_username
        user.password_hash = hash_password(new_password)
        db.commit()
        return {"message": "User updated successfully"}
    return {"message": "User not found"}

7. Main FastAPI Application (main.py)

Finally, your main FastAPI file (main.py) will be the entry point for the application and will call functions from user_api_function.py.

python
# main.py
from fastapi import FastAPI, Depends, HTTPException
from sqlalchemy.orm import Session
from .user_api_function import register_user, login_user, get_user, update_user
from .database import get_db
from .models import User
from .exceptions import InvalidCredentialsError

app = FastAPI()

@app.post("/register")
async def register(username: str, password: str, db: Session = Depends(get_db)):
    return register_user(db, username, password)

@app.post("/login")
async def login(username: str, password: str, db: Session = Depends(get_db)):
    return login_user(db, username, password)

@app.get("/user/{user_id}")
async def get_user_data(user_id: int, db: Session = Depends(get_db)):
    user = get_user(db, user_id)
    if not user:
        raise HTTPException(status_code=404, detail="User not found")
    return {"username": user.username}

@app.put("/user/{user_id}")
async def update_user_data(user_id: int, username: str, password: str, db: Session = Depends(get_db)):
    return update_user(db, user_id, username, password)
